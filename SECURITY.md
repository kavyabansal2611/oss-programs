## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please report it to us privately.

**Do not open a public issue.**

Instead, please:

1. Open a security advisory on GitHub using the [Security Advisories](https://github.com/OWASP-STUDENT-CHAPTER/oss-programs/security) feature
2. Or email the maintainers directly with details about the vulnerability

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 48 hours and keep you informed of our progress.

## Security Best Practices

When contributing to this project:

- Never commit sensitive information (API keys, credentials, etc.)
- Keep dependencies up to date
- Follow secure coding practices
- Report any security concerns promptly

Thank you for helping keep this project secure!
